Instrucciones para subir los acertijos

// 1. Clonar repositorio en PC local
- Instala Git Bash si no lo tienes instalado en tu PC 
- Ejectua 'git clone https://github.com/ramsestone/Save-The-World-Scape-Room' en el directorio donde quieres guardar el repositorio

// 2. Agrega tu acertijo, coloca pistas e instrucciones en sus archivos correspondientes. Rellena el "disco duro" del director con archivos de relleno (algunos archivos se pueden abrir y otros están corruptos). Si tu acertijo ocupa algún archivo real de imagen/audio/video colócalo donde creas necesario. Recuerda que las pistas deben llevar al usuario a dicho archivo y debe ser muy intuitivo de sacar la contraseña. La contraseña puede ser alfanumérica pero debe ser fácil de identificar.
- No te estanques en acertijos de juegos de palabras o de puro texto. Puedes usar archivos de audio, video, imágenes y lo que se te ocurra.
- Recuerda que el acertijo debe ser coherente al lore del juego que puedes encontrar en el archivo Lore.md
- Las pistas deben estar redactadas como si fuera información que los espías han logrado recabar.
- Las instrucciones son más para información entre nosotros. Para saber cómo llegar a cada contraseña 

// 2. Actualizar repositorio de GitHub
Una vez que tengas tu acertijo implementado
- git add .
- git commit -m "[Breve descripción de lo que estás subiendo]"
- git push origin main
